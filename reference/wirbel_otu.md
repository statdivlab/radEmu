# mOTU table from Wirbel et al. meta-analysis of fecal metagenomes.

A matrix with counts of mOTUs (metagenome-derived OTUs) for each sample.

## Usage

``` r
wirbel_otu
```

## Format

A matrix with mOTU counts.

- otu_table:

  OTU table with 566 samples and 845 taxa
