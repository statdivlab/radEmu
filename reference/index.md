# Package index

## All functions

- [`control_fn()`](https://statdivlab.github.io/radEmu/reference/control_fn.md)
  :

  Create list of control options (to pass to
  [`emuFit()`](https://statdivlab.github.io/radEmu/reference/emuFit.md))

- [`dpseudohuber_median_dx()`](https://statdivlab.github.io/radEmu/reference/dpseudohuber_median_dx.md)
  [`dpsuedohuber_median_dx()`](https://statdivlab.github.io/radEmu/reference/dpseudohuber_median_dx.md)
  : Calculate the derivative to the pseudo-Huber smoothed median

- [`emuFit()`](https://statdivlab.github.io/radEmu/reference/emuFit.md)
  : Fit radEmu model

- [`emuFit_check()`](https://statdivlab.github.io/radEmu/reference/emuFit_check.md)
  :

  Runs checks for appropriate arguments before running
  [`emuFit()`](https://statdivlab.github.io/radEmu/reference/emuFit.md)

- [`emuFit_micro()`](https://statdivlab.github.io/radEmu/reference/emuFit_micro.md)
  : Fit radEmu model. Called by emuFit

- [`emuFit_micro_penalized()`](https://statdivlab.github.io/radEmu/reference/emuFit_micro_penalized.md)
  : Fit radEmu model with Firth penalty

- [`fit_null()`](https://statdivlab.github.io/radEmu/reference/fit_null.md)
  : fits model with B_kj constrained to equal g(B_k) for constraint fn g

- [`fit_null_scc()`](https://statdivlab.github.io/radEmu/reference/fit_null_scc.md)
  : fits model with B_kj constrained to equal g(B_k) for constraint fn
  g, for a single category constraint

- [`fit_null_symmetric()`](https://statdivlab.github.io/radEmu/reference/fit_null_symmetric.md)
  : fits model with B_kj constrained to equal g(B_k) for constraint fn
  g, for a symmetric constraint

- [`fit_null_symmetric_subset()`](https://statdivlab.github.io/radEmu/reference/fit_null_symmetric_subset.md)
  : fits model with B_kj constrained to equal g(B_k) for constraint fn
  g, for a symmetric constraint

- [`make_design_matrix()`](https://statdivlab.github.io/radEmu/reference/make_design_matrix.md)
  :

  Generates the design matrix for a given `formula` and `data` object

- [`make_diagnostics_plots()`](https://statdivlab.github.io/radEmu/reference/make_diagnostics_plots.md)
  : makes plots to investigate convergence of estimation under null
  hypothesis

- [`make_reference_constraints()`](https://statdivlab.github.io/radEmu/reference/make_reference_constraints.md)
  : Make lists of constraint functions and their gradients when using a
  reference taxon

- [`plot(`*`<emuFit>`*`)`](https://statdivlab.github.io/radEmu/reference/plot.emuFit.md)
  : Plotting function

- [`power_model`](https://statdivlab.github.io/radEmu/reference/power_model.md)
  : A fitted GLM model for the power of radEmu to reject a truly false
  null hypothesis

- [`print(`*`<emuFit>`*`)`](https://statdivlab.github.io/radEmu/reference/print.emuFit.md)
  : Print function

- [`pseudohuber_median()`](https://statdivlab.github.io/radEmu/reference/pseudohuber_median.md)
  [`psuedohuber_median()`](https://statdivlab.github.io/radEmu/reference/pseudohuber_median.md)
  : Calculate the pseudo-Huber smoothed median

- [`score_test()`](https://statdivlab.github.io/radEmu/reference/score_test.md)
  : Run robust score test

- [`simulate_data()`](https://statdivlab.github.io/radEmu/reference/simulate_data.md)
  : Data simulation function

- [`wirbel_otu`](https://statdivlab.github.io/radEmu/reference/wirbel_otu.md)
  : mOTU table from Wirbel et al. meta-analysis of fecal metagenomes.

- [`wirbel_otu_small`](https://statdivlab.github.io/radEmu/reference/wirbel_otu_small.md)
  : subset of mOTU table from Wirbel et al. meta-analysis of fecal
  metagenomes.

- [`wirbel_sample`](https://statdivlab.github.io/radEmu/reference/wirbel_sample.md)
  : Covariates from Wirbel et al. meta-analysis of fecal metagenomes.

- [`wirbel_sample_small`](https://statdivlab.github.io/radEmu/reference/wirbel_sample_small.md)
  : Covariates from Wirbel et al. meta-analysis of fecal metagenomes,
  for a subset of samples.

- [`wirbel_taxonomy`](https://statdivlab.github.io/radEmu/reference/wirbel_taxonomy.md)
  : Taxonomy information for mOTUs from Wirbel et al. meta analysis.
