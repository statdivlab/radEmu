# Print function

Print function

## Usage

``` r
# S3 method for class 'emuFit'
print(x, n = 20, ...)
```

## Arguments

- x:

  Object of class `emuFit`

- n:

  The number of coefficient estimates to be printed (ordered by largest
  absolute value to smallest)

- ...:

  No optional arguments are accepted at this time.

## Value

`NULL`. Displays printed model summary.
